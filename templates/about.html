🎓 Smart Campus Event Management System

Smart • Secure • Digital

A modern digital platform developed to simplify college event registration, QR attendance, certificate generation and analytics.

🎯 Our Mission

Our mission is to simplify the management of college events by providing a secure, efficient and user-friendly digital platform for students and administrators.

🎯 Project Objectives

  • Digitalize college event management.
  • Allow students to register online.
  • Generate certificates automatically.
  • Provide QR-based attendance.
  • Display event analytics through dashboards.

💻 Technologies Used

🐍 Python Flask

Backend framework used to develop the web application.

🗄 MySQL

Stores student, admin, event and registration details.

🌐 HTML & CSS

Used to create responsive webpages and professional user interface.

🎨 Bootstrap

Provides responsive design and attractive components.

The Smart Campus Event Management System is developed to digitalize the management of college events. It allows students to register for events, view their registrations, download certificates, and helps administrators manage events efficiently.
